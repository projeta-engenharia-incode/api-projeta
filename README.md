# 🏗️ API - Projeta Engenharia

Este repositório será o ponto central de desenvolvimento da **API da Projeta Engenharia**, criada pela equipe da **Incode - Tech School**.

## 🚀 Sobre o Projeto

A API tem como objetivo fornecer uma base sólida e escalável para atender às necessidades da empresa **Projeta Engenharia**, utilizando tecnologias modernas e boas práticas de desenvolvimento backend.

## 🛠️ Tecnologias Base

- Java (versão 17+)
- Spring Boot

## 👨‍💻 Desenvolvimento

Este projeto está sendo desenvolvido por alunos e mentores da **Incode - Tech School**, com foco em aprendizado prático e aplicação de conceitos profissionais de engenharia de software.
